第一网：基础红线

不到这四条，直接淘汰。不看业绩。

红线 规则 一句话
成立时间 不到 3 年不要 没经历过牛熊验证
现任经理 任职不到 2 年不要 业绩不是他做的
基金规模 不到 20 亿不要 有清盘风险，操作受限
风格漂移 行业变动率超 50% 不要 追热点，没定力
⚠️ 特殊标记
→ 规模超过 200 亿 → 亮黄灯，必须证明收益没被规模稀释
→ 成立 3-5 年 → 观察区，需补基金经理此前业绩
筛完第一网，剩下约 30%
03

第二网：经理真伪

区分"真能力"和"运气好"

3.1
Alpha：你到底有没有额外创造价值
怎么看：把基金收益拆开，看看扣掉市场涨跌后，剩下多少是经理自己赚的。
标准
✓
年化超额 ≥ 3%，且不是偶然撞上的（统计上显著）
→ 真有本事
⚠
年化超额 1%-3%
→ 有迹象，再往下看
✗
年化超额 < 1%，或不显著
→ 跟指数差不多，不值得付费
3.2
信息比率：你的超额稳不稳
超额赚到手，是稳稳当当，还是大起大落？
✓
信息比率 ≥ 0.5
→ 超额扎实，不是运气波动
⚠
0.3-0.5
→ 再看其他项
✗
< 0.3
→ 超额质量差，淘汰
3.3
熊市见真章
不看牛市的收益，看熊市能不能少亏。
近5年至少经历两次大跌。两次都跑赢同类，才算通过。
✓
两次大跌都跑赢，且平均多赚 2% 以上
→ 抗跌，真本事
⚠
一次跑赢一次跑输
→ 待观察
✗
两次都跑输
→ 风控不行，淘汰
3.4
你究竟擅长什么
看一件事：过去两年，前三大行业是不是一直那三个？
✓
核心行业连续出现 6 个季度以上，前三大占一半以上仓位
→ 能力圈清晰
⚠
核心行业偶尔变化
→ 待观察
✗
每隔两三个季度就换一套行业
→ 风格漂移，淘汰
3.5
你的买卖值不值
看他买入的股票接下来涨没涨，卖出的跌没跌。
统计最近两年每次调仓的方向对不对。
✓
正确率 ≥ 60%
→ 调仓在创造价值
⚠
50%-60%
→ 再看其他项
✗
< 50%
→ 乱动不如不动，淘汰
3.6
你自己买了吗
✓
经理自己持有 ≥ 300 万
→ 利益绑定，通过
⚠
100-300 万
→ 有绑定但不强
✗
不到 100 万或不披露
→ 你的钱他不上心，扣分
第二网通过者：能力真实、风格稳定、言行一致

第三网：持仓含金量

穿透净值，看底层到底买了什么

4.1
龙头纯度
龙头不是自封的。看业内营收前三才算。
前十大持仓里，龙头的总权重：
✓
≥ 60%
→ 一水的好公司
⚠
40%-60%
→ 龙头里有二线
✗
< 40%
→ 买的是什么？
4.2
赚钱能力：加权 ROE
持仓公司整体的净资产收益率
✓
≥ 15%，且三年波动不大
→ 实实在在的好生意
⚠
12%-15%
→ 尚可
✗
< 12%
→ 赚钱能力弱，淘汰
4.3
成长预期
未来两年，这些公司加总的利润增速能有多少？
✓
≥ 15%
→ 明天比今天更好
⚠
10%-15%
→ 温和增长
✗
< 10%
→ 停滞或衰退
4.4
行业景气度：五灯检查
看持仓行业当前处于什么位置
政策绿灯：有支持
需求绿灯：订单在涨
资本绿灯：行业在扩张
库存绿灯：库存低位
盈利绿灯：利润在加速
✓
五灯全亮
→ 通过
⚠
亮三四灯
→ 观察
✗
亮两灯及以下
→ 淘汰
第三网通过：底层资产扎实，买的是好东西

第四网：脆弱性测试

前面的都是"过去不错"，这一层问"明天会不会崩"

5.1 规模诅咒
规模两年翻了一倍？
超额收益同时往下掉？
任一答案"是" → 淘汰
→ 规模已侵蚀能力
5.2 拥挤警报
前十大重仓股，是不是大家都在买？
被 50 只以上基金抱团的股票，占比超过一半 → 淘汰
→ 一旦踩踏，无处可逃
5.3 最痛月份
过去 5 年，有没有单月跌超 15%？
跌的原因是什么？
✓
若因系统崩盘（如熔断）
→ 可谅解
✗
若无缘无故
→ 风控有漏洞，淘汰
5.4 回撤深度
近 5 年最大回撤
✓
< 20%
→ 安全
⚠
20%-30%
→ 注意
✗
≥ 30%
→ 持有体验极差，淘汰
5.5 性价比
每承受 1% 回撤，能换来多少收益？
✓
卡玛比率 ≥ 1.0
→ 划算
⚠
0.5-1.0
→ 勉强
✗
< 0.5
→ 性价比极低
第四网通过：未来崩盘概率低，持有体验好
