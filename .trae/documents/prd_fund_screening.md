# 产品需求文档 (PRD) - 淘金筛网高清重绘版

## 1. 产品概述
将「淘金筛网」基金筛选体系信息图进行高清矢量化重绘，保持原图麦肯锡蓝橙设计风格和完整布局结构，仅提升文字清晰度和打印级分辨率。输出可编辑HTML版本、PNG高清图及可打印PDF。

- **核心目标**：文字重绘 + 高清矢量化，所有字体统一为清晰印刷级（300-600dpi适配）
- **目标用户**：需要高质量基金筛选框架展示材料的投资者/理财顾问

## 2. 核心功能

### 2.1 功能模块
1. **主信息图页面**：完整还原7大章节的基金筛选框架内容
2. **导出功能**：支持PNG高清导出、PDF打印导出

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|-------------|
| 淘金筛网信息图 | Hero区域 | 标题、副标题、核心价值主张 |
| | 01章节：选确定性 | 五把筛子、五个追问 |
| | 02章节：基础红线 | 四条红线规则表 + 特殊标记说明 |
| | 03章节：经理真伪 | 6项评估指标(Alpha/信息比率/熊市/能力圈/调仓/自持) |
| | 04章节：持仓含金量 | 4项指标(龙头纯度/ROE/成长预期/景气度) |
| | 05章节：脆弱性测试 | 5项压力测试(规模诅咒/拥挤/最痛月/回撤/性价比) |
| | 06章节：配置格局 | 三层分级池(战略核心/战术增强/观察池) |
| | 附：五问速查卡 | 快速参考表格 |

## 3. 核心流程
用户打开HTML → 浏览完整筛选框架 → 可选择导出PNG/PDF或直接打印

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：麦肯锡蓝 (#003366 深蓝 / #0066CC 亮蓝)
- **辅助色**：橙色系 (#FF8C00 / #FFA500)
- **强调色**：绿色通过 ✓ / 黄色观察 ⚠ / 红色淘汰 ✗
- **背景**：浅灰白渐变，保持专业金融感
- **字体**：思源黑体/苹方为主，确保印刷级清晰度
- **布局**：垂直长图信息流，卡片式分区，分隔线明确
- **图标风格**：简洁线性图标，与原设计一致

### 4.2 页面设计概述
| 区域 | 布局 | 颜色 | 字体大小 | 特殊元素 |
|------|------|------|----------|----------|
| 标题区 | 居中大字+副标题 | 深蓝底+白字 | 36-48px | 放大镜视觉元素 |
| 章节标题 | 左侧编号+标题+分割线 | 蓝色编号块 | 28-32px | 橙色装饰线 |
| 内容卡片 | 白底卡片带阴影 | 白底+浅蓝边 | 14-16px正文 | 三色状态标识 |
| 表格区 | 规整表格布局 | 斑马纹或纯色 | 13-15px | 图标列 |
| 结尾区 | 深色背景总结 | 深蓝底 | 20-24px | 灯塔视觉元素 |

### 4.3 响应式
- **桌面优先**：固定宽度1200px居中，保证打印一致性
- **打印优化**：@media print 样式，隐藏不必要元素，分页控制

## 5. 技术约束
- 单HTML文件，内嵌CSS/JS，无外部依赖（除字体CDN）
- 支持浏览器原生打印→PDF
- 使用html2canvas或类似方案支持PNG导出
- 所有文字使用矢量字体，禁止图片文字
